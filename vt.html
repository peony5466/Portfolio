Veille Technologique

Qu'est-ce que veille technologique ?

C’est un processus visant à s’informer systématiquement sur un sujet dans le but de se tenir au courant des avancées technologiques, techniques, produits ou nouvelles récentes. Elle implique la recherche, la surveillance, la collecte, l’organisation et la diffusion de l’information.


Qu'est-ce que Flutter ?

Flutter est un framework open-source pour le développement d’applications mobiles, web et de bureau créé par Google. Il permet aux développeurs de créer des interfaces utilisateur réactives et performantes à l’aide d’une seule base de code pour les plates-formes Android, iOS, Web et de bureau.

Langage utilisé

Les applications Flutter sont écrites dans le langage DART

Nouveaux domaines d'application:

  • IFlutter pour les jeux: Le développement de jeux occasionnels et de jeux plus complexes avec Flutter devrait se développer.
  • Flutter pour les applications IoT: La création d'interfaces utilisateur pour les appareils IoT avec Flutter pourrait devenir plus courante.
  • Flutter pour les applications AR/VR: L'utilisation de Flutter pour la création d'expériences de réalité augmentée et virtuelle est un domaine prometteur.

Adoption par l'industrie:

  • Le nombre de développeurs Flutter actifs a augmenté de manière exponentielle ces dernières années.
  • Adoption par de grandes entreprises: De nombreuses grandes entreprises ont adopté Flutter pour le développement d'applications mobiles, comme Google, Airbnb, BMW et Toyota.
  • Large choix de bibliothèques et d'outils: Un large éventail de bibliothèques et d'outils tiers est disponible pour Flutter, couvrant différents domaines tels que la gestion d'état, la communication réseau et l'authentification.

Avantages d'utilisation

  • Développement multi-plateforme: Créez des applications natives pour Android et iOS avec un seul code source.
  • Performances élevées: Profitez de performances natives grâce à la compilation directe en code machine.
  • Richesse de widgets: Utilisez une large gamme de widgets prédéfinis pour créer des interfaces utilisateur intuitives et attractives.
  • Facilité d'apprentissage: Le langage Dart est facile à apprendre et à utiliser, même pour les débutants.
  • Communauté active: Bénéficiez d'une communauté large et active pour vous aider et vous soutenir.

2024

les phase les plus importantes

  • Flutter 3.20: Lancement officiel avec des améliorations majeures.
  • Dart 3.4.10: Version majeure avec des avancées du langage.
  • Nouvelles plateformes:
    • Flutter pour Linux : Version alpha pour le développement d'applications Linux.
    • Flutter pour Web : Version bêta publique pour le développement d'applications web.
  • Sortie de FlutterMaps.

2023

les phase les plus importantes

  • Flutter 3.3: Lancement officiel, avec le compilateur Kotlin K2 en version alpha, des annotations d'exigence d'acceptation et des types définitivement non nullables.
  • Sortie de Dart 2.17: Améliorations du langage Dart, notamment la prise en charge des génériques de type récursifs.
  • Flutter pour macOS: Version bêta publique pour le développement d'applications macOS avec Flutter.
  • Sortie de l'extension FlutterGPT.

2022

les phase les plus importantes

  • Flutter 3.0: Lancement officiel, avec des fonctionnalités pour les jeux occasionnels, la prise en charge d'Apple Silicon et des améliorations de performance pour Android.
  • Sortie de Dart 2.14: Nouvelles fonctionnalités du langage Dart, telles que les expressions de type et les opérateurs de collection.
  • Prise en charge des annotations sur les paramètres de type de classe.
  • Flutter for Windows: Version bêta publique pour le développement d'applications Windows avec Flutter.

2021

les phase les plus importantes

  • Flutter 2.0: Lancement officiel, apportant des améliorations de performance et de stabilité, de nouveaux widgets et une meilleure prise en charge de Material Design.
  • Sortie de Dart 2.12: Améliorations du langage Dart, notamment la prise en charge des types non-nullables par défaut.
  • Temps de construction plus court grâce à la compilation parallèle dans le backend JVM IR.
  • Annonce de Flutter Web: Permet de créer des applications web avec Flutter, en utilisant le même code que pour les applications mobiles.
